# RecordApp

Webapp pública de lista de la compra con memoria histórica.

## MVP

- listas por fecha
- items editables y marcables
- detección de repetidos
- análisis simple de frecuencia
- recordatorios básicos
- autenticación por email/password y Google
- base PWA instalable

## Stack

- Next.js App Router
- Supabase Auth
- Supabase Postgres + RLS
- Vercel

## Roadmap técnico corto

- MVP: webapp pública + memoria histórica + recordatorios en interfaz
- Fase 1.5: PWA instalable
- Fase 2: push notifications de recordatorios
